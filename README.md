## Como rodar localmente

1. Clone o repositório  
2. Instale as dependências com npm install  
3. Rode o projeto com npm run dev  
4. Acesse http://localhost:3000 no seu navegador

## Tecnologias usadas

- React  
- Vite  
- JavaScript (ES6+)  
- CSS Modules
